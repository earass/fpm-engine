In order to run the GUI, you need to install Flask module first. You can install using pip:

pip install Flask

Following are the files details:

app/static: folder contains images and pdf reports to be displayed on the front-end
app/templates: contains the html file for the front-end
app/__init__.py contains code for the web app GUI using Flask framework

run.py: runs the web app on http://127.0.0.1:5000/

requirements.txt: contains all the packages that need to installed prior to running this project

In order to run the code, just run the run.py file.
